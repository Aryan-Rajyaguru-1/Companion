# Companion AI Modes Documentation

## Overview

Companion offers two distinct ways to interact with AI models, each designed for different use cases and requirements. This documentation explains the differences between **Web Mode** and **Desktop Mode** to help you choose the best option for your needs.

## üåê Web Mode (Cloud AI Only)

### What is Web Mode?

Web Mode allows you to use Companion directly in your web browser without any downloads or installations. It provides access to powerful cloud-based AI models through a modern web interface.

### Key Features

- **No Installation Required**: Access Companion instantly through any modern web browser
- **Always Up-to-Date**: Automatically receive the latest features and improvements
- **Cloud AI Models**: Access to cutting-edge models like GPT-4, Gemini, Claude, and more
- **Cross-Platform**: Works on any device with a web browser (Windows, Mac, Linux, mobile)
- **Real-Time Streaming**: Get responses as they're generated
- **Modern Interface**: Beautiful, responsive design optimized for web use

### Supported AI Models (Cloud)

- **OpenAI**: GPT-4o, GPT-4 Turbo, GPT-3.5 Turbo
- **Google**: Gemini 2.5 Flash, Gemini Pro
- **Anthropic**: Claude 3.5 Sonnet, Claude 3 Haiku
- **DeepSeek**: DeepSeek R1 (Cloud version)
- **Mistral**: Mistral Large, Mistral 8x7B
- **Perplexity**: Sonar models with web search

### Limitations

- **Internet Required**: Requires a stable internet connection
- **No Local Models**: Cannot run AI models locally on your device
- **Cloud Dependency**: Relies on external AI services
- **Data Privacy**: Messages are processed by cloud providers
- **Usage Costs**: May incur costs based on cloud model usage

### Best For

- **Quick Testing**: Try Companion without commitment
- **Casual Users**: Occasional AI interactions
- **Mobile Access**: Using AI on phones/tablets
- **No Storage Concerns**: Don't want to download large model files
- **Latest Models**: Access to newest cloud AI models

## üíª Desktop Mode (Cloud + Local AI)

### What is Desktop Mode?

Desktop Mode is the full Companion application that you download and install on your computer. It provides access to both cloud AI models (when online) and local AI models (works offline).

### Key Features

- **Hybrid Capabilities**: Use both cloud and local AI models
- **Offline Functionality**: Run AI models locally without internet
- **Enhanced Privacy**: Your data stays on your device with local models
- **Better Performance**: Optimized desktop experience
- **Local Model Management**: Download and manage AI models locally
- **Advanced Features**: Full feature set including model switching, history management, and more

### Supported AI Models

#### Cloud Models (same as Web Mode)
- **OpenAI**: GPT-4o, GPT-4 Turbo, GPT-3.5 Turbo
- **Google**: Gemini 2.5 Flash, Gemini Pro
- **Anthropic**: Claude 3.5 Sonnet, Claude 3 Haiku
- **DeepSeek**: DeepSeek R1 (Cloud version)
- **Mistral**: Mistral Large, Mistral 8x7B
- **Perplexity**: Sonar models with web search

#### Local Models (via Ollama)
- **Llama 3.2 3B**: General conversation and assistance
- **DeepSeek R1 1.5B**: Advanced reasoning and step-by-step analysis
- **CodeGemma 2B**: Code generation and programming help
- **CodeQwen 1.5B**: Advanced coding tasks and explanations
- **Custom Models**: Add any Ollama-compatible model

### Requirements

- **Operating System**: Windows 10+, macOS 10.14+, or Linux
- **Storage**: 2-8 GB for local models (depending on models chosen)
- **RAM**: 8 GB recommended (4 GB minimum)
- **Processor**: Modern CPU (any processor from last 5 years)
- **Internet**: Optional (required only for cloud models and updates)

### Installation

1. **Download**: Get the installer from the official website
2. **Install**: Run the installer for your operating system
3. **Setup**: Configure cloud API keys (optional)
4. **Download Models**: Choose which local models to install
5. **Start Chatting**: Begin using both cloud and local AI

### Advantages

- **Privacy**: Local models keep your data on your device
- **Offline Use**: Work without internet connection
- **Performance**: Optimized for desktop use
- **Full Control**: Manage models, settings, and data locally
- **No Usage Limits**: Local models have no per-message costs
- **Customization**: Advanced settings and configurations

### Limitations

- **Installation Required**: Needs to be downloaded and installed
- **Storage Usage**: Local models require disk space
- **Setup Complexity**: May require some technical setup
- **Updates**: Manual updates (though auto-update is available)

### Best For

- **Power Users**: Advanced AI interactions and workflows
- **Privacy-Conscious**: Keep sensitive data local
- **Offline Work**: Use AI without internet
- **Cost-Conscious**: Avoid per-message cloud costs
- **Developers**: Code generation and technical tasks
- **Regular Users**: Daily AI interactions

## üéØ How to Choose

### Choose Web Mode if you:
- Want to try Companion quickly without commitment
- Primarily use cloud AI models
- Don't mind internet dependency
- Use multiple devices (including mobile)
- Don't want to install software
- Have limited storage space

### Choose Desktop Mode if you:
- Want the full Companion experience
- Need offline AI capabilities
- Value privacy and data control
- Use AI regularly for work or projects
- Want to minimize cloud usage costs
- Prefer desktop applications

## üöÄ Getting Started

### Web Mode

2. Navigate to the demo/testing page
3. Start chatting immediately

### Desktop Mode

2. Download the installer for your OS
3. Install and run Companion
4. Follow the setup wizard
5. Download your preferred local models
6. Start using both cloud and local AI
