Powerful Features

Discover everything that makes Companion the most advanced AI chat interface available today.

Local AI Models

Run AI models locally using Ollama for complete privacy and offline capability. Your data never leaves your machine.

Cloud AI Integration

Access powerful cloud models through OpenRouter API, including GPT-4, Claude, and more cutting-edge models.

Beautiful Interface

Modern, responsive design that works seamlessly across desktop and web platforms with dark/light themes.

Code Highlighting

Built-in syntax highlighting for code blocks with support for dozens of programming languages.

Chat History

Automatic conversation saving and loading with search functionality to find previous discussions.

Export Conversations

Export your conversations in multiple formats including Markdown, PDF, and plain text.
